Linguagem Python

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

Vantagens
  • Simplicidade: Python é conhecido por sua sintaxe simples e legibilidade, o que o torna uma excelente escolha para iniciantes e programadores experientes.
  • Versatilidade: Python é usado em uma variedade de domínios, incluindo desenvolvimento web, análise de dados, inteligência artificial, automação, entre outros.
  • Comunidade Ativa: Python possui uma vasta comunidade de desenvolvedores que contribuem com bibliotecas, frameworks e recursos educacionais.
  • Portabilidade: Python é suportado em várias plataformas, incluindo Windows, macOS e Linux.
Conhecimento necessario

Para entender este tutorial, é útil ter algum conhecimento básico de programação. Não é necessário ter experiência prévia com Python, mas uma compreensão geral de conceitos de programação será útil.

Hello World

Fazer o "hello world" em Python é simples. Aqui está um exemplo de código:

print("Hello, world!")
Estruturas Condicionais

Exemplo de estrutura condicional em Python:


                        if x > 5:
                         print("x é maior do que 5")
                        else:
                         print("x é menor ou igual a 5")
Loops

Exemplo de loop em Python: 


                        for i in range(5):
                         print(i)
Listas

Exemplo de lista em Python: 


                        frutas = ["maçã", "banana", "laranja"]
                         print(frutas[0])  # Saída: maçã
Dicionarios

Exemplo de dicionário em Python: 


                        pessoa = {"nome": "João", "idade": 30, "cidade": "São Paulo"}
                        print(pessoa["idade"])  # Saída: 30
NumPy

NumPy é uma biblioteca Python usada principalmente para computação numérica. Ela fornece estruturas de dados e funções para trabalhar com matrizes e arrays multidimensionais.

Exemplo de uso do NumPy: 


                        import numpy as np
                        
                        arr = np.array([1, 2, 3, 4, 5])
                        print(arr)  # Saída: [1 2 3 4 5]
Pandas

Pandas é uma biblioteca Python usada para manipulação e análise de dados. Ela fornece estruturas de dados e funções para trabalhar com dados tabulares e séries temporais

Exemplo de uso do Pandas


                        import pandas as pd

                        data = {'Nome': ['Alice', 'Bob', 'Charlie'], 'Idade': [25, 30, 35]}
                        df = pd.DataFrame(data)
                        print(df)
